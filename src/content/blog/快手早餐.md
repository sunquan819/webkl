很多上班族因为赶时间，经常不吃早餐或者随便应付。其实，一份营养丰富的早餐不需要很长时间，只要15分钟，就能做出既好吃又健康的早晨。今天分享5道快手早餐食谱，让你每天都能元气满满地出门。

## 1. 牛油果鸡蛋吐司（10分钟）

这是一道简单到不会失败的早餐，同时营养丰富，颜值也很高。

**食材：**
- 全麦吐司 2片
- 牛油果 半个
- 鸡蛋 1个
- 盐、黑胡椒、柠檬汁少许

**做法：**
1. 吐司放入烤箱或平底锅烤至金黄
2. 牛油果去核，用叉子压成泥，加柠檬汁、盐调味
3. 煎一个荷包蛋（或水波蛋）
4. 将牛油果泥涂在吐司上，放上煎蛋，撒黑胡椒

**营养亮点：** 牛油果富含健康脂肪和膳食纤维，搭配鸡蛋的优质蛋白，是完美的早餐组合。

## 2. 隔夜燕麦杯（5分钟+前一晚准备）

前一晚花5分钟准备，第二天早上直接就能吃，最适合完全没时间的早晨。

**食材：**
- 燕麦片 50g
- 牛奶或酸奶 150ml
- 蜂蜜 1勺
- 水果（蓝莓、香蕉等）适量
- 坚果（核桃、杏仁等）适量

**做法（前一晚）：**
1. 玻璃杯中放入燕麦片
2. 倒入牛奶或酸奶，加蜂蜜搅拌均匀
3. 盖上盖子，放入冰箱冷藏过夜

**早上：** 取出后放上切好的水果和坚果即可享用。

**营养亮点：** 燕麦是低GI食物，提供持久能量；水果和坚果补充维生素和健康脂肪。

## 3. 蔬菜蛋饼（12分钟）

中式早餐的经典选择，加足蔬菜让营养更均衡。

**食材：**
- 鸡蛋 2个
- 面粉 2勺
- 胡萝卜丝、葱花、菠菜适量
- 盐少许

**做法：**
1. 鸡蛋打散，加入面粉和少量水搅拌成糊
2. 加入切好的蔬菜和盐，搅拌均匀
3. 平底锅刷薄油，倒入蛋糊摊成饼
4. 中火煎至两面金黄即可

**营养亮点：** 一份蛋饼就包含蛋白质、碳水化合物和多种蔬菜，营养全面又扛饿。

## 4. 香蕉花生酱三明治（5分钟）

最快手的早餐，香甜软糯，小孩子也特别喜欢吃。

**食材：**
- 全麦面包 2片
- 花生酱 2勺
- 香蕉 1根
- 肉桂粉少许（可选）

**做法：**
1. 面包片涂上花生酱
2. 香蕉切片铺在一片面包上
3. 撒少许肉桂粉，盖上另一片面包
4. 对半切开即可

**营养亮点：** 花生酱提供蛋白质和健康脂肪，香蕉补充钾和快速能量。

## 5. 番茄鸡蛋面（15分钟）

热乎乎的汤面，是冬天早晨最温暖的选择。

**食材：**
- 挂面 1把
- 番茄 1个
- 鸡蛋 1个
- 葱花、盐、香油少许

**做法：**
1. 番茄切块，鸡蛋打散
2. 锅中放少许油，炒番茄至出汁
3. 加入适量水烧开，下入面条
4. 面条快熟时淋入蛋液，加盐调味
5. 撒葱花，滴几滴香油

**营养亮点：** 汤面暖胃易消化，番茄富含维C，鸡蛋补充优质蛋白。
