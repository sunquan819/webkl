很多人问过我：用AI是不是很贵？

说实话，付费版确实不便宜。ChatGPT Plus每月$20，Claude Pro也是$20，如果还要用API，那成本更高。

但是，完全免费也能用得很好。今天分享我的省钱经验。

## 我的主要省钱策略

我现在日常用的组合是这样的：

**日常对话** → Claude免费版，质量够用

**编程** → DeepSeek免费版，编程强

**实时信息** → Gemini免费版，能联网

**本地需求** → Llama本地跑，完全免费

**特殊任务** → 借朋友的GPT-4账号（开玩笑的）

基本上免费组合能覆盖90%的需求，剩下的10%才会考虑付费。

## 各个免费版的使用体验

### Claude免费版

Claude的免费版是我用得最多的。Claude 3 Sonnet这个模型质量很不错，写文章、整理文档、回答问题都能用。

限制是每天大概十几轮对话，用完了就提示升级。

但我的经验是，如果等几个小时再回来，有时候额度又刷新了。不是每次都行，但经常能再续一会儿。

注册账号需要邮箱，国内访问需要...你懂的。

### ChatGPT免费版

GPT-3.5确实比GPT-4差一些，但日常使用其实够用。

写简单文案、翻译文档、回答常识问题，GPT-3.5都能搞定。复杂推理、长文本处理这些确实不如GPT-4，但不是每个人都需要这些功能。

限制是每天有使用上限，但具体多少官方没说清楚。用多了会提示升级，第二天刷新。

### Gemini免费版

Gemini免费版的优势是能联网搜实时信息。

比如我问"最近有什么科技新闻"，它会自己去搜索，然后给你最新的信息。这个功能很实用。

缺点是中文有时候不够自然，复杂中文任务不如其他模型。

### DeepSeek免费版

DeepSeek是我最近的主力，特别是编程方面。

国内访问方便，不用折腾网络问题。免费版限制也比较宽松，不像Claude那么快就提示升级。

编程能力强，我问过一些挺复杂的代码问题，回答质量不错。

### 通义千问免费版

阿里的通义千问，中文能力确实强。

写中文内容、处理中文文档，比国外模型自然很多。免费版限制也宽松，日常用够用。

国内访问方便，不用网络问题。

## 一个账号不够？那就多账号

这个方法有点不好意思说，但确实有效。

Claude、ChatGPT这些免费版都有每日限制。如果你用量大，一个账号不够用。

我的做法是注册多个账号轮换。用完一个等一会儿换下一个，基本能覆盖一整天的需求。

当然要遵守平台规则，别太明显地刷，一般平台也不会太计较。

## API免费额度怎么薅？

如果你要自己开发一些AI应用，API免费额度就很重要了。

### DeepSeek API

DeepSeek新注册送500万tokens，这个额度真的很够用了。

就算收费，价格也便宜到几乎可以忽略。GPT-4的十分之一价格，质量还不错。

国内开发者首选这个，省钱又方便。

### Gemini API

Google给Gemini API提供了免费额度，每天有一定限制。

申请需要Google Cloud账号，过程有点繁琐但值得。

适合需要实时搜索能力的应用。

### OpenAI API

新注册送$5，有效期三个月。

问题是需要国外手机号验证，国内用户不太方便。而且三个月用不完就没了，有点坑。

### 其他平台

Together AI、OpenRouter这些平台都有免费额度或免费模型，可以多试试。

## 完全免费：开源模型本地部署

如果你不想受任何限制，开源模型本地部署是最好的选择。

### 我用的方法

我用Ollama，一个很简单的工具。

安装好后，一行命令就能跑模型：

```
ollama run llama3
ollama run qwen2
```

用起来和在网页上对话差不多，但完全免费，没有限制。

### 硬件需求

这个要看你电脑配置。

我的笔记本16GB内存，能跑Llama 3 8B和Qwen 7B这些中等大小的模型。

如果你有游戏显卡，可以跑更大的模型，效果更好。

没有好显卡也能用，就是慢一点。CPU跑7B模型大概几秒一个回答，能用。

### 哪些开源模型好用？

**Llama 3** - Meta的模型，综合能力强

**Qwen2** - 阿里开源，中文特别好

**Mistral** - 法国团队，性价比高

**DeepSeek Coder** - DeepSeek开源，编程强

本地跑这些，想用多少用多少，完全自由。

## 第三方平台也能白嫖

除了官方平台，还有些第三方平台提供免费AI。

### Poe

Quora旗下的Poe，每天能免费用一次各种模型，包括GPT-4。

虽然只能一次，但遇到特别需要GPT-4的任务时可以用用。

### HuggingChat

HuggingFace的免费聊天，提供多种开源模型。

完全免费，没有限制，模型种类也多。

### Perplexity

Perplexity的搜索功能免费可用，查资料很方便。

不像ChatGPT那样可能编信息，它给出真实来源。

## 我的省钱总结

整体来说，我的策略是：

**90%需求用免费版解决**

Claude免费版、DeepSeek免费版、通义千问，这些日常够用。

**特殊需求才用付费**

真的需要GPT-4级别能力的时候，才考虑付费。或者借朋友的账号。

**能本地部署就本地**

敏感数据、无限使用这些需求，本地部署最合适。

**API用DeepSeek**

如果要开发应用，DeepSeek API省钱又好用。

---

用AI不一定非得花钱。免费组合加开源部署，基本能满足大部分需求。

当然，如果你预算充足，付费版体验确实更好，省很多折腾。但像我这样精打细算，也完全能用好AI。

希望这些经验对你有帮助。有问题可以试试这些免费方案，够用就别急着付费。
