AI 쓰려면 돈 내야 한다는 거 아시죠? ChatGPT Plus 월 $20, Claude Pro 월 $20... 꽤 부담스러워요.

근데 꼭 돈 내야 할까요? 저는 1년 넘게 AI를 쓰면서 돈을 거의 안 썼어요. 오늘은 제 절약 노하우를 공유할게요.

## 제 AI 사용 철학

"무료로 충분하면 굳이 유료를?"

이게 제 철학이에요. 물론 유료가 더 좋은 건 맞아요. 하지만 제 용도엔 무료로 충분해요.

## 무료 AI 서비스 총정리

### Claude 무료

제가 가장 많이 쓰는 서비스예요.

**장점**
- 글쓰기 퀄리티가 높아요
- 긴 문서 처리가 가능해요
- 한국어가 자연스러워요

**제한**
- 하루에 약 10-20번 대화 가능
- Claude 3.5 Sonnet 사용 가능 (최상위 모델은 Pro만)

**팁**
- 아침에 접속하면 할당량이 많아요
- 대화를 한 번에 길게 하면 효율적이에요

### ChatGPT 무료

가장 유명한 서비스예요.

**장점**
- 안정적이에요
- GPT-3.5/4o mini 사용 가능
- 실시간 검색 (제한적)

**제한**
- GPT-4는 하루 몇 번만 가능
- 사용량이 제한적이에요

**팁**
- GPT-4 할당량은 아껴서 쓰세요
- 간단한 건 GPT-3.5로 충분해요

### Gemini 무료

구글의 서비스예요.

**장점**
- 실시간 검색이 잘 돼요
- 무료 할당량이 넉넉해요
- 구글 서비스와 연동돼요

**제한**
- 한국어가 조금 어색해요
- 복잡한 추론은 약해요

**팁**
- 최신 정보 검색할 때 쓰세요
- 한국어보다 영어로 물어보는 게 나을 수 있어요

### DeepSeek 무료

국내에서 접근 쉬운 서비스예요.

**장점**
- 접속이 쉬워요 (국내 직접 접속 가능)
- 무료 할당량이 넉넉해요
- 코딩에 강해요

**제한**
- 영어가 메인이라 한국어가 약간 떨어져요
- 실시간 검색이 안 돼요

**팁**
- 코딩 작업에 쓰세요
- API도 무료로 쓸 수 있어요

### 통합 서비스

여러 AI를 한 곳에서 쓸 수 있는 서비스도 있어요.

**Poe**
- 여러 모델을 한 곳에서 사용
- GPT-4, Claude 등을 무료로 조금씩 사용 가능
- 하루 사용 제한이 있어요

**HuggingChat**
- 완전 무료
- 오픈소스 모델 사용
- 제한이 거의 없어요

## 제 사용 전략

저는 이렇게 나눠서 써요.

### 글쓰기: Claude

글 쓸 때는 Claude가 제일 자연스러워요. 블로그 글, 이메일, 보고서 등을 쓸 때 써요.

하루 할당량이 있어서, 중요한 글 쓸 때 아껴뒀다가 써요.

### 코딩: DeepSeek

코딩은 DeepSeek이나 Claude를 써요. DeepSeek은 무료 할당량이 넉넉해서 많이 쓸 수 있어요.

### 검색: Gemini

최신 정보가 필요할 때 Gemini를 써요. 실시간 검색이 돼서 편해요.

"오늘 뉴스 뭐 있어?", "이 회사 주가 어때?" 같은 질문에 좋아요.

### 간단한 작업: ChatGPT

간단한 질문, 번역, 요약은 ChatGPT 무료로 충분해요.

## 한 서비스가 막히면?

하루 할당량을 다 쓰면 다른 서비스로 넘어가요.

오전: Claude → 할당량 소진
오후: DeepSeek 또는 Gemini
저녁: ChatGPT 또는 다른 서비스

이렇게 하면 하루 종일 AI를 쓸 수 있어요.

## 로테이션 전략

여러 계정을 쓰는 것도 방법이에요. (물론 정책상 권장하진 않아요.)

저는 그냥 여러 서비스를 쓰는 걸로 충분했어요. 굳이 계정을 여러 개 만들지 않아도, 서비스만 잘 나눠도 충분해요.

## API 무료로 쓰기

개발자라면 API를 무료로 쓸 수도 있어요.

### DeepSeek API

가입하면 500만 토큰을 무료로 줘요. 꽤 많은 양이에요.

### Gemini API

Google Cloud 계정으로 무료 할당량을 받을 수 있어요.

### OpenAI API

신규 가입하면 $5를 줘요. (3개월 유효)

### Together AI, OpenRouter

이런 플랫폼에서도 무료 모델을 제공해요.

## 완전 무료: 로컬 실행

진짜 완전히 무료로 쓰고 싶다면, 로컬에서 실행하세요.

### Ollama

설치하면 로컬에서 모델을 실행할 수 있어요.

```
ollama run llama3
ollama run qwen2
```

이렇게 하면 완전 무료, 무제한이에요.

### 장점

- 완전 무료
- 무제한 사용
- 인터넷 필요 없음
- 데이터가 로컬에 있어서 안전

### 단점

- 컴퓨터 사양이 좋아야 해요
- 최신 모델은 어려워요
- 설정이 조금 복잡해요

## 유료가 필요한 순간

그래도 유료가 필요할 때가 있어요.

### 1. 업무로 쓸 때

시간이 돈인데, 할당량 때문에 기다릴 수 없을 때는 유료가 낫아요.

### 2. 최상위 모델이 필요할 때

GPT-4, Claude 3.5 Sonnet 같은 최상위 모델은 유료만 쓸 수 있어요.

### 3. 데이터 보안이 중요할 때

기업 데이터를 다룰 때는 공짜 서비스를 쓰면 안 돼요.

### 4. API를 많이 쓸 때

개발용으로 API를 많이 호출하면 유료가 필요해요.

## 제 절약 성과

1년 동안 AI에 쓴 돈은 $0예요. 그래도 제 업무, 블로그, 코딩 다 AI 도움을 받았어요.

돈을 안 써서가 아니라, 무료로 충분해서 안 쓴 거예요.

## 팁 정리

1. **여러 서비스를 병행하세요** - 하나가 막히면 다른 걸 쓰세요.
2. **용도별로 나누세요** - 글쓰기는 Claude, 검색은 Gemini, 코딩은 DeepSeek.
3. **할당량을 아끼세요** - 중요한 작업에 아껴두세요.
4. **로컬 실행도 고려하세요** - 개발자라면 로컬 실행도 좋아요.
5. **API 무료 할당량을 활용하세요** - 개발할 때 유용해요.

## 마치며

AI를 꼭 돈 내야만 잘 쓸 수 있는 건 아니에요. 무료 서비스도 충분히 강력해요.

물론 유료가 더 편하고, 더 많은 기능을 제공하는 건 맞아요. 하지만 개인 사용, 학습, 취미용이라면 무료로 충분해요.

저처럼 여러 무료 서비스를 잘 활용해 보세요. 생각보다 훨씬 많은 걸 할 수 있어요.

돈 내지 않아도 AI를 잘 쓸 수 있어요. 그게 제 결론이에요.
